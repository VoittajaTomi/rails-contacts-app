# README

An Ruby on Rails Demo app. You can sign up and add your private contact details of people.

Installation details:

- Clone
- Install
```
   cd contacts
   bundle install
   rake db:migrate
   rails s
   
   ```

* Ruby version
`2.3.1`
* System dependencies:
  - rbenv, sqlite3
  - gems:
    - Rails 5.0.1
